# Guia_Latex_Grabación

Latex no es un lenguaje de programación, es un sistema (no trabaja bajo una logica) de marcado de información. Brinda estructura a traves de una serie de instrucciones. Al compilar entrega un resultado entrega un documento.

 - Libros
 - Tesis
 - Informes

Los compiladores nos permiten converitr las instrucciones que le di para hacer el documento. Aunque cumplen la misma función, unos son mas eficientes con ciertas cosas pero con otras no.

**Compilador Overleaf:**

Templates (o plantillas): El alcance que realmente tiene Latex y lo que se puede hacer (casi que hay de todo). No es solo para articulos, hay para libros, informes; hay también plantillas de otras instituciones educativas que pueden ser usadas, etc.

 - Presentaciones
 - Posters
 - Calendarios
 - Hojas de vida
 - Cartas formales
 - Reportes

Overleaf tiene una limitante en cuanto a peso (lo mas probable), mas que por contenido, es más versatil en cuanto a Word (pero mas que todo en el campo de las ciencias duras) debido a que todo esta enfocado en circuitos, ecuaciones, graficas, tablas, etc.
